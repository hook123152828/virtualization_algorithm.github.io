演算法原理
使用最小堆（優先佇列）貪心取最近節點，依序確定各節點最短距離。只適用非負邊權。
最短路貪心Priority Queue非負邊
虛擬碼
初始化: dist[src]=0，其餘=∞
pq = [(0, src)] // 最小堆
while pq 非空:
(d, u) = pq.pop()
if visited[u]: continue
visited[u] = true
for each (u→v, w):
if dist[u]+w < dist[v]: 更新並入堆
return dist[]
