演算法原理
dp[i] 為以 arr[i] 結尾的 LIS 長度。對每個 i 枚舉 j < i 且 arr[j] < arr[i] 取最大。
LISDP
虛擬碼
dp[i] = 1 // 以 arr[i] 結尾的 LIS
for i=1 to n-1:
for j=0 to i-1:
if arr[j]<arr[i]: dp[i]=max(dp[i],dp[j]+1)
return max(dp[])
