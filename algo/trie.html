演算法原理
多叉樹結構，每個節點代表一個字元，從根到葉的路徑對應一個字串。支援 O(|w|) 插入與查詢。
字串前綴樹字典
虛擬碼
root = {}
insert(word):
node = root
for c in word:
if c not in node: node[c]={}
node = node[c]
node.end = true
search(word):
node = root
for c in word:
if c not in node: return false
node = node[c]
return node.end
