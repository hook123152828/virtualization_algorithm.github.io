演算法原理
在標準 DSU 上額外記錄各連通塊的大小，union 時合併計數。
並查集連通塊大小
虛擬碼
初始化: parent[i]=i, sz[i]=1
find(x): 路徑壓縮取根
union(a, b):
ra=find(a), rb=find(b)
if ra==rb: return // 已連通
parent[rb] = ra
sz[ra] += sz[rb] // 合併塊大小
maxBlock = max(sz[find(i)] for all i)
