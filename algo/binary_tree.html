演算法原理
前序（根→左→右）、中序（左→根→右）、後序（左→右→根）三種 DFS 走訪。
二元樹DFS前序/中序/後序
虛擬碼
preorder(node): // 前序
visit(node); left; right
inorder(node): // 中序
left; visit(node); right
postorder(node): // 後序
left; right; visit(node)
if node == null: return
