演算法原理
在輾轉相除的同時回推 x, y 使得 ax + by = gcd(a,b)，可用於求模逆元或解線性同餘。
數論GCD模逆元
虛擬碼
exgcd(a, b) → (g, x, y):
if b==0: return (a, 1, 0)
(g, x1, y1) = exgcd(b, a%b)
x = y1
y = x1 - floor(a/b)*y1
return (g, x, y)
