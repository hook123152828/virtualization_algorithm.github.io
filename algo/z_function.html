演算法原理
Z[i] 表示 s[i..] 與 s 的最長公共前綴長度。利用已知匹配視窗 [l,r] 避免重複比較。
字串前綴匹配Z算法
虛擬碼
Z[0] = n // 整串與自身
l=0, r=0 // 已知匹配視窗
for i=1 to n-1:
if i < r: Z[i] = min(r-i, Z[i-l])
extend: 字元比較延伸 Z[i]
if i+Z[i]>r: l=i, r=i+Z[i]
return Z[]
