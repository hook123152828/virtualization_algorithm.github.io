演算法原理
每件物品可選無限次。正序更新允許重複選取：dp[j] = max(dp[j], dp[j-w]+v)。
背包DP完全背包正序更新
虛擬碼
dp[0..W] = 0 // 初始化
for each item(w, v):
for j = w to W: // 正序！
dp[j] = max(dp[j], dp[j-w]+v)
return dp[W]
