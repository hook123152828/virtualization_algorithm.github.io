演算法原理
dp[i][j] = LCS(A[0..i-1], B[0..j-1])，當字元相等則取對角線+1，否則取上方/左方最大值。
LCSDP二維DP
虛擬碼
初始化: dp[0][j]=dp[i][0]=0
for i=1..m, j=1..n:
if A[i-1]==B[j-1]: dp[i][j]=dp[i-1][j-1]+1
else: dp[i][j]=max(dp[i-1][j], dp[i][j-1])
return dp[m][n]
