#21 · 路徑壓縮 + 按秩合併

速度
演算法原理
路徑壓縮 + 按秩合併，使 find() 均攤接近 O(1)。維護連通分量的集合代表元。
並查集路徑壓縮按秩合併連通分量
虛擬碼
初始化: parent[i]=i, rank[i]=0
find(x):
if parent[x]≠x: parent[x]=find(parent[x])
return parent[x] // 路徑壓縮
union(a, b):
ra=find(a), rb=find(b)
if rank[ra]<rank[rb]: swap(ra,rb)
parent[rb]=ra // 按秩合併
