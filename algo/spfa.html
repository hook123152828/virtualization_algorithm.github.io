演算法原理
佇列優化的 Bellman-Ford，只對距離更新過的節點重新鬆弛鄰邊，平均效率較佳。
最短路負邊權Queue優化
虛擬碼
初始化: dist[src]=0，其餘=∞
queue = [src], inQueue[src] = true
while queue 非空:
u = queue.dequeue(); inQueue[u]=false
for each (u→v, w):
if dist[u]+w < dist[v]:
dist[v] = dist[u]+w
if !inQueue[v]: queue.push(v)
return dist[]
