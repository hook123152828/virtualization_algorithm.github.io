演算法原理
在有序序列上定義判斷函式 check(x)，使序列具有「前半全 True、後半全 False」的單調性。每次取中點，根據 check 結果將搜尋範圍減半，直到 L = R 收斂至邊界位置。
有序序列O(log n)邊界搜尋二分法
虛擬碼 find_min_false
L = 0, R = n-1
while L < R:
mid = L + (R-L) / 2
if check(mid): // arr[mid] < target
L = mid + 1 // 向右縮
else:
R = mid // 向左縮
return L // 第一個 False 的位置
L = 0, R = n-1
while L < R:
mid = L + (R-L+1) / 2 // 防死循環
if check(mid): // arr[mid] < target
L = mid // 向右縮
else:
R = mid - 1 // 向左縮
return L // 最後一個 True 的位置
