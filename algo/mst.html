演算法原理
排序邊後貪心選取，用 DSU 判斷是否形成環，選入不形成環的最小邊直到有 n-1 條。
最小生成樹貪心DSU排序
虛擬碼
排序所有邊（權重升冪）
MST = [], DSU 初始化
for each edge(u,v,w):
if find(u) ≠ find(v): // 不成環
union(u, v); MST.add(edge)
if |MST| == n-1: break
return MST
