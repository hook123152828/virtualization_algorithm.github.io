#07 · 點修改 + 前綴和查詢，O(log n)

格式：modify i val 或 query r（1-indexed，query 回傳 prefix sum[1..r]）

速度
演算法原理
利用二進位低位元（lowbit）組織前綴和。點修改與前綴查詢各 O(log n)，實作極簡。
前綴和點修改lowbit
虛擬碼
lowbit(x) = x & (-x)
update(i, delta):
while i≤n: tree[i]+=delta; i+=lowbit(i)
query(r): // 前綴和 [1,r]
s=0
while r>0: s+=tree[r]; r-=lowbit(r)
return s
