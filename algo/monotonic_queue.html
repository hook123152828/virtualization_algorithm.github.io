演算法原理
雙端佇列維護視窗內的最大/最小值候選，保持佇列單調性，O(n) 解決滑動視窗問題。
滑動視窗雙端佇列最大/最小值
虛擬碼
dq = [] // 存下標，佇列頭最大
for i=0 to n-1:
while dq 非空 && a[dq.back()]≤a[i]:
dq.pop_back() // 維護遞減
dq.push_back(i)
if dq.front() ≤ i-k: dq.pop_front()
if i≥k-1: result[i-k+1]=a[dq.front()]
