演算法原理
每個合數只被其最小質因數標記一次，嚴格 O(n) 時間篩出所有質數。
質數O(n)篩法線性篩
虛擬碼
is_prime[] = true; primes = []
for i=2 to n:
if is_prime[i]: primes.push(i)
for p in primes:
if i*p > n: break
is_prime[i*p] = false
if i%p==0: break // 保證唯一標記
return primes
