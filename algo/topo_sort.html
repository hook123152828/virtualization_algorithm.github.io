演算法原理
BFS 版：計算所有節點入度，入度為 0 者入佇列，依序輸出並減少鄰接節點入度。
有向無環圖BFS入度
虛擬碼
計算各節點入度 in_deg[]
將入度=0節點加入佇列
while queue 非空:
u = queue.dequeue()
result.push(u)
for each (u→v):
in_deg[v]--
if in_deg[v]==0: queue.push(v)
return result // 或判斷是否有環
