演算法原理
分治結構，每個節點維護區間聚合值。Lazy 標記延遲下傳，支援 O(log n) 區間修改與查詢。
區間查詢區間修改Lazy Propagation
虛擬碼
build(node, l, r):
if l==r: tree[node]=arr[l]; return
build(2n,l,mid); build(2n+1,mid+1,r)
tree[node] = tree[2n]+tree[2n+1]
update(l,r,v): 區間加法 + lazy
若節點完全覆蓋: lazy+=v; tree+=v*(r-l+1)
否則: push_down; 遞迴; push_up
query(l,r): 區間查詢
若節點完全覆蓋: return tree[node]
否則: push_down; 合併左右子樹
