演算法原理
按需開點，只在實際訪問到的節點才分配記憶體。適合值域極大但操作稀疏的情況。
動態開點線段樹稀疏
虛擬碼
nodes 陣列，cnt=1（根）
update(node, l, r, pos, val):
if l==r: node.val+=val; return
mid=(l+r)/2
if pos≤mid: 若無左子則開點; update(left...)
else: 若無右子則開點; update(right...)
push_up(node)
query(node, l, r, ql, qr):
若完全覆蓋: return node.val
否則合併左右子樹查詢
